# Evaluation

This directory contains evaluation notebooks, test data, and analysis results for the RAG system.

## Files

- **`hotpotqa_1000_samples.json`** - HotPotQA dataset samples used for evaluation
- **`hotpotqa_test_sample_and_vectorization.ipynb`** - Notebook for sampling and vectorizing HotPotQA data
- **`prepare_test_responses.ipynb`** - Notebook for preparing test responses from the RAG system
- **`test_responses.csv`** - Generated test responses from the RAG pipeline
- **`RAG_Evaluation_with_Ragas_+_Ollama.ipynb`** - RAGAS evaluation notebook using Ollama
- **`EvaluationResults.csv`** - Complete evaluation results with RAGAS metrics
- **`EvaluationResultsAnalysis.ipynb`** - Analysis and visualization of evaluation results

## Evaluation Pipeline

1. Sample and vectorize HotPotQA data
2. Generate responses using the RAG system
3. Evaluate with RAGAS framework
4. Analyze and visualize results
